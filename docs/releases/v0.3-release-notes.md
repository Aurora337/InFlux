# InFlux v0.3 Release Notes

## Purpose
Validate that multiple validators reach consistent agreement on state hashes under normal operation.

## Features Added
- Multi-node consensus simulator for validator rounds.
- Consensus verification integration with validator node abstractions.
- Agreement-rate and divergence tracking outputs.

## Validation Results
- Consensus suite reached full pass rate in ideal scenarios.
- Agreement rate measured at 1.000 in validated baseline runs.

## Known Limitations
- Adversarial and degraded conditions were not fully exercised in this milestone.
- Cross-environment determinism comparison was not part of v0.3 scope.

## Next Milestone
v0.4 Resilience Testing: inject faults and validate recovery while preserving replay guarantees.
