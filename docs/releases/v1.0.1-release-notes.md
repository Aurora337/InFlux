# InFlux v1.0.1 Release Notes

## Purpose
Add audit log with SHA-256 hash tracking for pipeline artifacts.

- Audit log generation with SHA-256 hashes for each artifact.
- Deterministic hash chain linking pipeline stages.

## Validation Results
- Audit log hashes verified against artifact content.
- Hash chain integrity confirmed.

## Known Limitations
