# InFlux v1.1.0 Release Notes

## Purpose
Add release attestation validation pipeline combining reproducibility generation, attestation generation, and cross-artifact consistency checks.

## Features Added
- One-command pipeline: reproducibility generation + release attestation generation/validation.
- Cross-artifact consistency verification between pipeline outputs.
- Deterministic integrity scoring for the combined pipeline.

## Validation Results
- Pipeline produces consistent results across repeated runs.
- Cross-artifact hashes match expected values.
- Release attestation validation passes end-to-end.

## Known Limitations
- Pipeline validation (v1.1.1) provides independent verification of pipeline outputs.
