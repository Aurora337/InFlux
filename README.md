# In~Flux — Deterministic Economic Engine

In~Flux is a fully deterministic economic protocol built on a unique 
Reproduction–Reserve–Circulation model with mathematically guaranteed
stability, scarcity control, and institutional-grade auditability.

## Components
- Deterministic Math Kernel (A–H)
- Economic Kernel: R(t), ΔC, ESM, scarcity/decay
- Ledger Kernel: 7-stage CTOR pipeline
- Synchronization Kernel: GDE, ROS, SHCM, AINF
- Alignment Kernel: 17 routing pathways
- Multi-Node Architecture: VN, SN, REN, LN, AN, PTN, FI
- Test Harness Integration
- Prototype + Validator Testnets
